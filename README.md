# CircleDemo
仿微信实现的朋友圈，模拟与后台交互实现了点赞、评论、删除等功能，listview可以根据键盘的显示或隐藏实现联动。与后台交互采用mvp模式。

图片显示规则类似微信朋友圈的排列
